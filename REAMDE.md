# Shopping Cart Checkpoint

## Description
This project is a dynamic shopping cart created using HTML, CSS, JavaScript, Bootstrap, and Font Awesome. 
Users can:
- Increase or decrease product quantity
- Remove products from the cart
- Like products with a heart icon
- See the total price updated automatically

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap 5
- Font Awesome

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Interact with the cart by clicking +, -, trash, or heart icons
